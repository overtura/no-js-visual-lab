선언형 Popover

JavaScript 없음

이 패널은 HTML 속성만으로 열리고 닫힙니다. 스타일과 위치는 CSS가 담당합니다.

Exclusive Accordion

<details name>으로 같은 그룹 안에서 하나의 항목만 열리게 합니다.

구조

HTML이 상태와 접근성 기본 동작을 맡고 CSS는 표현에 집중합니다.

시각

container query와 :has()를 더하면 컴포넌트가 스스로 배치와 강조를 바꿉니다.
