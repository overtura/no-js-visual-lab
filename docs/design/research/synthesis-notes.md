# Synthesis Notes

## Adopted Principles
- no-JS interaction을 먼저 보이고 설명은 짧게 둔다.
- 기능별 section과 tokenized controls를 사용한다.
- 상태 변화는 색상 외에도 border, contrast, text로 표현한다.

## Rejected Principles
- framework playground 구조는 no-JS 제약과 맞지 않는다.
- 외부 브랜드 visual identity는 복제하지 않는다.

## Token Changes
- `design-system/base.css`에 light native lab token을 둔다.

## Golden Screens
- Hero with orbit motif
- Popover demo open state
- `:has()` toggle state
